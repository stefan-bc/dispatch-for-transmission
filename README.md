# Dispatch for Transmission

A dark-mode Chrome extension for controlling a remote Transmission daemon.

> Third-party extension. Not affiliated with, endorsed by, or an official
> product of The Transmission Project. "Transmission" is a trademark of
> its respective owners; the name is used here only to describe
> compatibility (nominative fair use).

## Install (unpacked)

1. In Chrome, open `chrome://extensions`.
2. Enable **Developer mode** (top right).
3. Click **Load unpacked** and select this folder.
4. Pin the extension icon so it's visible in the toolbar.
5. Click the icon → "Open settings" to add your first server.

## Server setup

In Transmission: **Preferences → Remote** → enable remote access, set a
username/password (recommended), and note the host/port (default `9091`).

In Dispatch for Transmission → **Add server**:

- **Name** – any label you like.
- **RPC URL** – usually `http://HOST:9091/transmission/rpc`.
- **Web UI URL** – optional, for the "Open Web UI" link.
- **Require authentication** – tick if you set a username/password.
- **Download directories** – optional shortcuts that appear in the right-click
  menu (e.g. Movies → `/downloads/movies`).

Click **Test connection** first; the extension will request host permission
for the URL's origin.

## Keyboard

| Key | Action |
|---|---|
| `O` | Add torrent |
| `Space` | Start / pause selected (toggles) |
| `Del` / `⌫` | Delete selected (Enter confirms) |
| `S` or `/` | Focus search |
| `↑` `↓` / `J` `K` | Move selection (hold `Shift` to extend) |
| `Cmd/Ctrl`+`A` | Select all visible |
| `Enter` | Open file picker for focused torrent |
| `Esc` | Close dialog / clear search |
| `Cmd/Ctrl`+`Shift`+`P` | Open / close the popup (global) |

## Right-click menus

- **On a page, on any magnet link or `.torrent` URL** — "Add torrent to
  &lt;server&gt;" entry per configured server. Download-directory
  shortcuts appear as sub-entries.
- **On a torrent row in the popup** — Resume, Pause, Remove, Show files,
  plus Add / Start all / Stop all.
- **On empty space in the popup** — Add torrent, Start all, Stop all.

## Drag and drop

Drop a `.torrent` file anywhere on the popup to add it to the active server.

## Toolbar badge

The extension icon shows a count of fully-downloaded torrents for the
last-used server. Toggle it off in the settings dialog.

## Notes

- Dracula dark theme via DaisyUI v5. The extension is MV3 and uses
  `optional_host_permissions`, so Chrome only asks for access to servers
  you actually add.
- Passwords are stored in `chrome.storage.local` (not synced).
- Icon generated by `tools/make_icon.py` (pure-stdlib PNG writer).

## Rebuilding the stylesheet

`css/app.css` is a one-shot Tailwind v4 + DaisyUI v5 build. The **extension
runtime has no JS/CSS dependencies** — `app.css` is pre-compiled. If you
change HTML class names or the `build/input.css` theme:

```
cd build
npm install
npm run build   # writes ../css/app.css
```

`build/node_modules/` is not checked in and not needed at runtime.
